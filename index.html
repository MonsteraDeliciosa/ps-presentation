Other shortcuts

  • Shift + (tool shortcut) - change tools

  • Space - pan

  • Ctrl - move

  • Ctrl + R - toggle rulers

  • Ctrl + ; - toggle guidelines

  • Ctrl + ‘ - toggle grid

  • Ctrl + D - deselect

  • Ctrl + Shift + I - reverse selection

  • Ctrl + Alt + I - image size

  • Ctrl + Alt + C - canvas size
